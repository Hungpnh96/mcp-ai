Music · ZingMp3 proxy

Thử nghiệm và điều khiển phát nhạc tương thích ESP32.

Dùng chung API `/stream_pcm`, `/proxy_audio`, `/proxy_lyric` với thiết bị thực. Kết quả trả về bám sát chuẩn adapter.
